# proyecto diplomado

### 1. clonar el proyecto
```bash
git clone https://github.com/elmer200987/diplomado.git
```
### 2. instalar dependencias
```bash
npm install
```
### 3 copiar variables de entorno y configurar
```bash
cp env.sample .env
```
### 4 levantar el proyecto
### 4.1 En desarrollo
```bash
npm run dev
```
### 4.2 en produccion 
```bash
npm run start
```
